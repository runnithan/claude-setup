# Use /copy to Copy Claude's Output With Full Formatting Preserved

## TL;DR

/copy puts the last response in your clipboard with markdown formatting intact—no manual selection needed.

## Why it matters

Manually selecting and copying Claude's output often strips formatting or captures partial content. /copy captures the entire last response with markdown preserved, ready to paste into Notion, GitHub, or any markdown-aware tool.

## How to apply

After Claude generates output you want to use: type `/copy` and press Enter. The response is now in your clipboard. For agent teams: each agent's output can be copied individually by navigating to that agent pane and using /copy. Works in all session types including worktrees and forked sessions.
