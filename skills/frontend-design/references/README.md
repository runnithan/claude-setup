# Design References

Place examples of your best frontend designs in this directory.

The frontend-design skill uses these as reference material when generating new designs. Including past designs helps maintain consistency and quality by showing concrete examples of successful patterns.

## What to add

- Screenshots of completed designs (PNG/JPG)
- HTML/CSS/JSX snippets of standout components
- Color palettes and typography pairings that worked well
- Layout patterns worth reusing

## How it's used

When generating new designs, the skill can reference these files to understand your preferred aesthetic, color choices, and layout patterns — producing output that feels consistent with your existing work rather than generic.
